
Para a utilização deste programa é necessário ter o módulo "nltk" instalado.
Notas sobre a instalação do mesmo podem ser encontradas aqui:
http://www.nltk.org/install.html

Após a instalação do "nltk", é necessário copiar o ficheiro "pt_grammar.fcfg" para a pasta "/usr/share/nltk_data/".
(Caso o directorio não exista, o mesmo deverá ser criado.)

Para utilizar apenas é necessário fazer "import" do ficheiro "dcg.py".
